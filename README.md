# 🎮 Real-Time Game Analytics Platform

## 🚀 Overview

Built a real-time data pipeline to simulate and analyze game user behavior using Kafka, PostgreSQL, and Metabase.

## 🏗️ Architecture

Producer → Kafka → Consumer → PostgreSQL → Metabase Dashboard

## ⚙️ Tech Stack

- Python
- Apache Kafka
- PostgreSQL
- Docker
- Metabase

## 📊 Features

- Real-time event streaming
- User session tracking
- Revenue analytics
- Player behavior analysis

## 📈 Insights

- Revenue is dominated by a small group of high-spending users (whales)
- High engagement does not always translate to revenue
- Session duration varies significantly across users
